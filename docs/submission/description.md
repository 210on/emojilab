# 学会作品集向け文案

## タイトル

EmojiLab.

## 短い作品説明文

EmojiLab. は、チャットアプリ向け文字絵文字の制作と視認性評価を統合した研究支援システムである。ライト / ダーク背景での見え方を確認しながら、文字装飾の調整とPNG出力を一貫して行える。

## さらに短い版

文字絵文字の制作と視認性評価を統合し、見え方を確認しながら直感的に調整できる研究支援システム。

## 研究寄りの説明文

本作品は、チャットアプリ向け文字絵文字のデザイン支援を目的とした研究システムである。APCAコントラスト、縮小耐性、AIによる文字複雑性診断を同一画面に統合し、視認性評価とUI操作を往復しながら絵文字デザインを改善できる点に特徴がある。

## 作品集キャプション候補

- 文字絵文字の見え方を、制作と評価の両面から支援するシステム
- 視認性評価を内包した文字絵文字デザイン支援インタフェース
- 小さく表示される絵文字のための制作・評価統合環境
